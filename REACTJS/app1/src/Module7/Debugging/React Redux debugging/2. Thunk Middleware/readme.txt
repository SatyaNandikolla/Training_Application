1. Install redux & react-redux: npm install redux react-redux

2. Install redux logger: npm install redux-logger

3. Install redux thunk: npm install redux-thunk

4. Install redux devtools: npm install --save redux-devtools-extension

5. Create a folder "react-modules" in "<project>/src" folder.

6. Create a folder "components" in "<project>/src" folder.

7. Copy "App.js" into your "<project>/src/App.js".

8. Run the Reactjs project - "npm start".
