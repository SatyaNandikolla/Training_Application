1. Install enzyme and enzyme-adapter: npm install --save-dev enzyme @wojtekmaj/enzyme-adapter-react-17

2. Copy the folder "components" in "<project>/src" folder.

3. Copy "App.js" into your "<project>/src/App.js".

4. Copy "App.test.js" into your "<project>/src/App.test.js".

5. Run the Reactjs project - "npm start".
