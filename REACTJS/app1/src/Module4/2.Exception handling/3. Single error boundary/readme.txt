1. Copy "App.js" into your "<project>/src/App.js".

2. Create a folder "components" in "<project>/src" folder.

3. Copy "components/ErrorBoundary.js" into your "<project>/src/components" folder.

4. Copy "components/Todo_Function.js" into your "<project>/src/components" folder.

5. Copy "components/Todo_Class.js" into your "<project>/src/components" folder.

6. Run the Reactjs project in development mode- "npm start".

To run the server in production mode:
	- npm run build
	- serve -s build
