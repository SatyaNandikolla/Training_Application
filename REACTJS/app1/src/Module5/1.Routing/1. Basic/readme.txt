1. Install react-router package in <project> folder: npm install react-router

2. Install react-router-dom package in <project> folder: npm install react-router-dom --save

3. Copy "App.js" into your "<project>/src/App.js".

4. Create a folder "components" in "<project>/src" folder.

5. Copy "components/Header.js" into your "<project>/src/components" folder.

6. Copy "components/Footer.js" into your "<project>/src/components" folder.

7. Copy "components/Home.js" into your "<project>/src/components" folder.

8. Copy "components/Todo_Function.js" into your "<project>/src/components" folder.

9. Copy "components/Todo_Class.js" into your "<project>/src/components" folder.

10. Run the Reactjs project - "npm start".

11. Try urls:
	- http://localhost:3000/
	- http://localhost:3000/func
	- http://localhost:3000/class
